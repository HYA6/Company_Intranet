## ✔ 추후 유지 보수 및 기능 추가

* ### 유지 보수
  * 발견된 문제점
    * 1회성 조건문이나 필요없는 예외 처리 등 지저분한 코드
    * 정의되지 않은 DB 연관 관계
    * 사진 첨부 없이 식단표 작성 시 500 오류 발생
  * 해결 방안
    * 전체적인 코드 분석 후 정리
    * DB 외래키에 따른 CASCADE 및 제약 조건 설정

* ### 배포
  * 배포 관련하여 공부 후 적용 예정

* ### 기능 추가
  * 쪽지 기능
  * 메일 송수신 기능
  * 경영진 및 관리자가 사용할 수 있는 사원 관리 시스템
    * 구성원 추가 및 삭제
    * 조직도 작성
    * 연차 설정
  * 연봉 테이블 계산
